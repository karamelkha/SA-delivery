# Основные сущности и ключевые атрибуты:

## 1. Таблица: **Customer**

| Поле       | Тип данных   | Описание                   | Ключ |
| ---------- | ------------ | -------------------------- | ---- |
| CustomerID | INT          | Идентификатор пользователя | PK   |
| Name       | VARCHAR(100) | Имя клиента                |      |
| Phone      | VARCHAR(20)  | Телефон                    |      |
| Email      | VARCHAR(100) | Электронная почта          |      |
| Address    | TEXT         | Адрес доставки             |      |

## 2. Таблица: **Restaurant**

| Поле         | Тип данных   | Описание                | Ключ |
| ------------ | ------------ | ----------------------- | ---- |
| RestaurantID | INT          | Идентификатор ресторана | PK   |
| Name         | VARCHAR(100) | Название ресторана      |      |
| Phone        | VARCHAR(20)  | Телефон                 |      |
| Email        | VARCHAR(100) | Электронная почта       |      |
| Address      | TEXT         | Адрес ресторана         |      |

## 3. Таблица: **MenuItem**

| Поле         | Тип данных    | Описание                 | Ключ |
| ------------ | ------------- | ------------------------ | ---- |
| MenuItemID   | INT           | Идентификатор блюда      | PK   |
| RestaurantID | INT           | Внешний ключ на ресторан | FK   |
| Name         | VARCHAR(100)  | Название блюда           |      |
| Description  | Text          | Описание блюда           |      |
| Price        | Decimal(10,2) | Цена блюда               |      |

## 4. Таблица: **Order**

| Поле         | Тип данных    | Описание                       | Ключ |
| ------------ | ------------- | ------------------------------ | ---- |
| OrderID      | INT           | Идентификатор заказа           | PK   |
| RestaurantID | INT           | Внешний ключ на ресторан       | FK   |
| CustomerID   | INT           | Внешний ключ на пользователя   | FK   |
| OrderDate    | DateTime      | Дата и время оформления заказа |      |
| Status       | VARCHAR(50)   | Статус заказа                  |      |
| TotalPrice   | Decimal(10,2) | Итоговая цена заказа           |      |

## 5. Таблица: **OrderItem**

| Поле        | Тип данных    | Описание                         | Ключ        |
| ----------- | ------------- | -------------------------------- | ----------- |
| OrderItemID | INT           | Идентификатор позиции заказа     | PRIMARY KEY |
| OrderID     | INT           | Внешний ключ на заказ            | FOREIGN KEY |
| MenuItemID  | INT           | Внешний ключ на блюдо            | FOREIGN KEY |
| Quantity    | INT           | Количество                       |             |
| Price       | DECIMAL(10,2) | Цена за единицу на момент заказа |             |

---

## 6. Таблица: **Courier**

| Поле        | Тип данных   | Описание                   | Ключ        |
| ----------- | ------------ | -------------------------- | ----------- |
| CourierID   | INT          | Идентификатор курьера      | PRIMARY KEY |
| Name        | VARCHAR(100) | Имя курьера                |             |
| Phone       | VARCHAR(20)  | Телефон курьера            |             |
| VehicleType | VARCHAR(50)  | Тип транспортного средства |             |

---

## 7. Таблица: **Delivery**

| Поле           | Тип данных  | Описание                         | Ключ        |
| -------------- | ----------- | -------------------------------- | ----------- |
| DeliveryID     | INT         | Идентификатор доставки           | PRIMARY KEY |
| OrderID        | INT         | Внешний ключ на заказ            | FOREIGN KEY |
| CourierID      | INT         | Внешний ключ на курьера          | FOREIGN KEY |
| DeliveryStatus | VARCHAR(50) | Статус доставки (напр. "В пути") |             |
| DeliveryTime   | DATETIME    | Время доставки                   |             |

---

## Связи между таблицами

| Откуда         | Куда           | Тип связи                      | Описание                                  |
| -------------- | -------------- | ------------------------------ | ----------------------------------------- |
| **Customer**   | **Order**      | один-ко-многим                 | Один клиент может сделать много заказов   |
| **Restaurant** | **MenuItem**   | один-ко-многим                 | В ресторане много блюд                    |
| **Order**      | **OrderItem**  | один-ко-многим                 | Заказ состоит из нескольких позиций       |
| **Order**      | **Delivery**   | один-ко-одному                 | Каждый заказ связан с одной доставкой     |
| **Courier**    | **Delivery**   | один-ко-многим                 | Курьер может выполнять множество доставок |
| **Order**      | **Restaurant** | многие-ко-одному (через Order) | Заказ относится к конкретному ресторану   |

---
