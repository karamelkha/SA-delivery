# SA-delivery

Проектирование информационной системы сервиса доставки еды 'Delivery'

Цель проекта: Разработать концепцию и основные требования для создания новой информационной системы, автоматизирующей процесс обработки заказов, управления меню и координации доставки между ресторанами, курьерами и клиентами.

Моя роль: Системный аналитик.

Ключевые задачи:

1. Разработка требований.
2. Моделирование бизнес-процессов (To-Be).
3. Проектирование и моделирование данных.
4. Прототипирование интерфейсов.
