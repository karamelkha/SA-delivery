# Traceability Matrix — Delivery

Документ описывает трассировку бизнес-правил (BR), функциональных требований (FR),
нефункциональных требований (NFR), пользовательских сценариев (Use Case)
и бизнес-процессов (BPMN).

Таблица используется для проверки полноты и согласованности требований,
для оценки влияния изменений на систему, а также для подтверждения соответствия требований бизнес-логике и процессам.
